# Certificado de Crecimiento Exponencial

Calculadora especializada para el Certificado de Crecimiento Exponencial de Cosecha Capital, que utiliza el Interés Compuesto Multiplicador™ para multiplicar activos reales productivos.

## Características

- **Interés Compuesto Multiplicador™**: No solo multiplica dinero, multiplica activos reales productivos
- **Configuración de metas del inversionista**: Define tu objetivo de ingreso pasivo
- **Plan de retiros personalizado**: Diseña tu estrategia de retiros año por año
- **Análisis de impacto amplificado**: Visualiza cómo la reinversión multiplica el impacto positivo
- **Reportes HTML especializados**: Genera reportes profesionales con información completa
- **Soporte multimoneda**: MXN, USD, EUR con tipos de cambio actualizables
- **Gráficos interactivos**: Visualización avanzada del crecimiento patrimonial

## Tecnologías

- React 18 con TypeScript
- Tailwind CSS para estilos
- Framer Motion para animaciones
- Recharts para gráficos
- Vite como bundler

## Desarrollo

```bash
npm run dev
```
